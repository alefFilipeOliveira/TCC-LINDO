
UPAMED - Final package
Installation:
1. Copy UPAMED_final folder to C:\xampp\htdocs\UPAMED_final
2. Open http://localhost/UPAMED_final/index.php
3. Login: EnfermeiroUpa2 / Upa2 (or use demo link)
Notes:
- Symptoms selection: click 'Adicionar' on catalog to add symptoms (no free-text)
- Report includes BMI (IMC) calculation and an extended simulated analysis
- Download PDF button available on the report page (client-side using jsPDF)
- For real server-side AI, place OPENAI_API_KEY in config.php
